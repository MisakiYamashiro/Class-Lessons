Seçici Türleri

Satır bazında CSS(inline)

Etiket seçiciler

Temel de bütün sayfa veya bir grup içerisindeki aynı etikete sahip tüm içeriklerin etkilenmesi için kullanılır. head bloğuna style etiketi tanımlanır ve seçiciler bu etiket içerisine yazılır. CSS özellikleri {} küme parantezi içerisine yazılarak birden çok CSS özelliğinin tanımlanması sağlanır.

Sınıf Seçiciler.

Önce sınıf tanımlanır. Body içerisinde aynı özellikte olması istenen tüm etiketlere class attribute'u ile sınıf ataması yapılır

ID Seçiciler.

ID, HTML sayfasında unique olarak atanmış veridir/etikettir. CSS ile bu unique elemanın özelliklerini değiştirebilirz.
